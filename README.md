# Jenga 3D

3D-игра Jenga на базе **React + Vite + Three.js** (через `@react-three/fiber` и `@react-three/drei`).

## Возможности текущей версии

- Классическая башня: **18 слоёв × 3 блока = 54 блока**.
- Чередование ориентации слоёв на 90°.
- Выбор блока кликом.
- Ход по правилам: выбранный блок переносится наверх.
- Базовая проверка устойчивости и проигрыш при падении (упрощённая логика).
- Перезапуск партии.
- Улучшенный визуал:
  - вариации оттенка дерева,
  - контуры граней блоков,
  - улучшенное освещение (ambient/hemisphere/directional/point),
  - фон и лёгкий туман сцены.
- Оптимизация загрузки:
  - code splitting в Vite,
  - lazy loading `GameScene`.

## Запуск проекта

### 1) Установка зависимостей

```bash
npm install
```

### 2) Режим разработки

```bash
npm run dev
```

По умолчанию приложение будет доступно по адресу:

`http://localhost:5173/`

### 3) Production-сборка

```bash
npm run build
```

### 4) Локальный просмотр production-сборки

```bash
npm run preview
```

## Игровая логика (текущая реализация)

1. Игрок выбирает блок кликом.
2. Нажимает **«Сделать ход»**.
3. Блок переносится на верх башни:
   - если верхний слой полный — создаётся новый слой,
   - иначе блок добавляется в верхний неполный слой.
4. Проверяется устойчивость башни.
5. При критическом смещении фиксируется проигрыш.

---

## Ограничения текущей версии

- Используется базовая физическая модель (rigid body), но без тонкой настройки масс/трения под настольный реализм.
- Детекция проигрыша основана на порогах наклона/просадки, а не на полном анализе устойчивости.
- Управление через кнопку хода, без drag-and-drop.
